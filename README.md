# VK Telegram Parsing

Pipelines for parsing, collecting, and normalizing open data from VK and Telegram.

## Overview
This repository contains tools and experiments for collecting and structuring open platform data from VK and Telegram.

## Goals
- collect open data
- normalize entities and metadata
- support analytics and research workflows
- build reusable parsing pipelines

## Scope
- VK parsing
- Telegram parsing
- data normalization
- export pipelines

## Stack
- Python
- APIs / parsers
- ETL workflows
